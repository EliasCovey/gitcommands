 - clone: used to create a copy of a specific repository or branch within a repository

 - checkout: lets you navigate between the branches created by the git branch command

     + -b: create a new branch and immediately navigates there

 - status: shows the current state of your git directory and staging area.

 - push: used to upload local reposity content to a remote repository.   

    + -u: pushes the previous branch you were on

 - pull: The command used to update the repository with new changes

 - add: Add contents of new or changed files to the index

 - commit:  Creates a save point in your project; records a snapshot of your files at a certain time

    + -m: commits changes with a message

    + -a: commit all tracked changes

    + -am: commits all tracked changes with a message

 - branch: Lists the existing branches

    + -a: lists both local and remote branches

    + -d: delete the given branch

 - merge: Merge a branch into your current branch
